# Cheezza 🍕

Cheezza, ASP.NET Core MVC mimarisi kullanılarak geliştirilmiş bir pizza dükkanı yönetim ve sipariş web uygulamasıdır.

## 🚀 Kullanılan Teknolojiler
* **Backend:** C#, ASP.NET Core MVC
* **Veritabanı:** SQLite & Entity Framework Core
* **Güvenlik:** ASP.NET Core Identity (Kayıt, Giriş ve Rol Yönetimi)
* **Frontend:** HTML, CSS, Bootstrap

## ✨ Özellikler
* **Kullanıcı Kimlik Doğrulaması:** Güvenli kayıt olma ve giriş yapma (Login/Register) sistemi.
* **Rol Bazlı Yetkilendirme:** Sadece "Admin" rolüne sahip kullanıcıların erişebildiği güvenli yönetim paneli.
* **CRUD İşlemleri:** Admin panelinden yeni pizza ekleme, listeleme, detay görüntüleme, güncelleme ve silme işlemleri.

## 🛠️ Kurulum ve Çalıştırma
1. Projeyi bilgisayarınıza klonlayın.
2. Visual Studio 2022 ile `Cheezza.sln` dosyasını açın.
3. Projeyi çalıştırdığınızda (F5) SQLite veritabanı otomatik olarak oluşturulacaktır.
